VTP - Első módszer (vlan database)

Switch# vlan database Switch(vlan)# vtp domain tartomanynev Switch(vlan)# vtp password jelszo Switch(vlan)# vtp v2-mode Switch(vlan)# vtp mode server Switch(vlan)# vtp mode client Switch(vlan)# vtp mode transparent

VTP - Második módszer (globális konfiguráció)

VTP ellenőrzése

Switch# show vtp status Switch# show vtp password

VTP Pruning

Switch(config)# vtp pruning

A VTP lehetővé teszi a VLAN információk automatikus szinkronizálását a trönk kapcsolatokon keresztül. Server módban VLAN létrehozható és terjeszthető, Client módban csak fogadja a módosításokat, Transparent módban nem módosítja a VTP adatbázist, de továbbítja a VTP kereteket. A pruning csökkenti a fölösleges VLAN forgalmat a trönkökön.
