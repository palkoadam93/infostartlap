Modulok, csomagok

Feladatok

1. Írj egy programot, amely importálja a math modult, és használja a math.factorial() függvényt, hogy kiszámolja egy adott szám faktoriálját!

2. Készíts egy Python programot, amely importálja a random modult, és generál egy véletlenszerű egész számot 1 és 100 között, majd írd ki a számot!

3. Írj egy programot, amely importálja a datetime modult, és jelenítse meg az aktuális dátumot és időt a datetime.datetime.now() függvény használatával!

4. Készíts egy Python programot, amely importálja a os modult, és használja az os.listdir() függvényt egy adott könyvtár tartalmának felsorolására!

5. Írj egy programot, amely importálja a json modult, és használja a json.dumps() függvényt egy Python szótár JSON formátumú szöveggé alakítására!

6. Készíts egy Python programot, amely importálja a requests csomagot, és használja a requests.get() függvényt egy weboldal tartalmának letöltésére és az első 100 karakter kiírására!

7. Írj egy programot, amely importálja a pandas csomagot, és használja a pandas.DataFrame() függvényt egy adatkeret létrehozására néhány mintaadat segítségével!

8. Készíts egy Python programot, amely importálja a re modult, és használja a re.findall() függvényt egy szövegben található összes email cím keresésére és kiírására!

9. Írj egy programot, amely importálja a numpy csomagot, és használja a numpy.array() függvényt egy egyszerű tömb létrehozására és a tömb elemeinek kiírására!

10. Készíts egy Python programot, amely importálja a scipy csomagot, és használja a scipy.optimize.minimize() függvényt egy egyszerű optimalizálási probléma megoldására, például egy függvény minimumának megtalálására!
